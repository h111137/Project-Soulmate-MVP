"단 한 장의 사진과 터치로 탄생하는, 나만을 위한 도파민 파트너"

1. Project Overview

기존 AI 채팅 서비스의 복잡함을 제거하고, '동물상 테스트'의 간편함과 바이럴 요소를 결합한 가장 쉬운 AI 컴패니언 서비스입니다.

2. Why This Service?

Problem: 기존 서비스는 설정이 어렵고 영어권 중심이라 한국인의 감성을 채워주지 못함.

Solution: 심리 테스트 기반의 원클릭 캐릭터 생성 + 멀티모달(음성/이미지) 인터랙션 제공.

3. Core Features (MVP)

[ ] Personality Test: MBTI 기반 성향 분석 알고리즘 (Python)

[ ] Character Gen: Stable Diffusion API 연동을 통한 실시간 이미지 생성

[ ] Voice Call: 특정 시간대 푸시 알림 및 보이스 메시지 전송

4. Tech Stack Strategy (AI-Native)

Frontend: HTML/CSS/JS (Generated by Cursor)

Backend: Python Flask/FastAPI (Generated by Claude)

AI Ops: Prompt Engineering을 통해 개발 속도 10배 가속화
